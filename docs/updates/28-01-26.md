## 14:00 (SGT) - Meeting Scheduling with Manual Links

Added ability for users to enter their own meeting link when scheduling.

**Changes:**
- Added meeting link input field to `ScheduleMeetingModal`
- Added automatic browser timezone detection using `Intl.DateTimeFormat`
- Updated `handleScheduleSubmit()` to send `duration`, `meetingLink`, and `timezone`
- Meeting times stored in UTC and displayed in user's local timezone

**Files Modified:**
- `src/components/circles/ScheduleMeetingModal.jsx`
- `src/pages/Circles.jsx`

**Data Flow:**
1. User enters title, selects time, pastes meeting link
2. Frontend detects timezone, converts time to UTC
3. API receives: `{title, scheduledAt, duration, meetingLink, timezone}`
4. Other users see "Join Call" button with the link
