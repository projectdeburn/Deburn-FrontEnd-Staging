# Session Log - 19/02/26

## Fix "Everyone Declined" Bug in Meeting Scheduling

Meeting scheduling was marking all attendees as "declined" regardless of availability selection.

**Problem:** The `schedule_meeting` service compared `availableMembers` names (sourced from `circlegroups.members[].name`) against `users.firstName + " " + users.lastName`. The formats never matched, so no member was ever found as "available".

**Fix:** Changed `meeting_service.py` to use `m.get("name")` from the group member array directly — the same source as the availability endpoint.

## Recurring Meetings — Full Feature Implementation

Added full recurring meeting support with per-user skip tracking. Uses a single-document approach — no separate documents per occurrence.

### Design Decisions
- **Single document:** One meeting doc with `recurrence: true` + computed occurrences (no proliferation of meeting docs)
- **Computed occurrences:** `compute_upcoming_occurrences()` generates future dates from the anchor `scheduledAt` — pure computation, no DB calls
- **Per-user skip tracking:** `skippedOccurrences: [{userId, date}]` array on the meeting document
- **Backwards compatible:** Old docs without `recurrence` field are treated as non-recurring via `.get("recurrence")` returning falsy

## Bug Fix: Recurrence Not Stored in MongoDB

Meetings were saved as `recurrence: false` despite the toggle being on.

**Problem:** `Circles.jsx:handleScheduleSubmit` destructured only `title`, `scheduledAt`, `duration`, `meetingLink`, `timezone`, `availableMembers` — it never passed `recurrence` or `frequency`.

**Fix:** Added both fields to the API payload:
```javascript
recurrence: meetingData.recurrence || false,
frequency: meetingData.frequency || undefined,
```

Changed from hiding skipped occurrences to showing them with visual strikethrough.

**Frontend:** `GroupDetailsModal.jsx` applies `.group-details-meeting-date--skipped` class (line-through + lighter color) to skipped occurrences. "Not this time" button targets the first non-skipped occurrence.
