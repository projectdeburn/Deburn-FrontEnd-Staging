# Updates - 30/01/26

## Article Image Support for Micro-Courses

### Overview

Added full article image support to the Learning/Micro-Courses feature, including image fetching, display, and loading animations.

### Features Added

1. **Article Image Fetching** - Images fetched from `/api/article-image/{id}/{lang}`
2. **Blur-to-Sharp Transition** - Images load with smooth blur effect
3. **Loading Overlay** - Spinner shown until all content (including images) loads
4. **White Modal Background** - Clean white background for article reading
5. **Image Sizing** - Constrained to max 600x350px, centered

### Files Modified

| File | Changes |
|------|---------|
| `src/components/learning/ArticleModal.jsx` | Image fetching, loading states, content visibility |
| `src/prototype.css` | Image styles, loading overlay, modal background |

### New CSS Classes

- `.article-loading-overlay` - Loading overlay with spinner
- `.article-loading-spinner` - Animated spinner
- `.article-image` - Article image container with vignette
- `.text-content-body.visible` - Fade-in animation

### API Integration

Uses new backend endpoint:
- `GET /api/article-image/{content_id}/{language}` - Returns binary image data

### Response Fields Used

```json
{
  "hasImage": true,
  "hasImageEn": true,
  "hasImageSv": true
}
```
