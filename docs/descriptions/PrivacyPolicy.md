# Privacy Policy

## Overview

The Privacy Policy page displays Eve's privacy policy, explaining how user data is collected, used, and protected. It's a static content page accessible to all users.

## Components

- Eve logo link
- Page title and last updated date
- Content sections:
  - Introduction
  - Information We Collect
  - How We Use Your Information
  - Data Security
  - Your Rights
  - Contact Us
- Back to Eve link

## User Flow

1. User clicks Privacy Policy link from Login, Register, or footer
2. Reads privacy policy content
3. Scrolls through sections
4. Clicks Back to Eve to return to app
